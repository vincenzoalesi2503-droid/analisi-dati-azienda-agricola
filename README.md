# Analisi Dati Aziendali – KPI, Vendite e Produzione (Python) 

## Descrizione
Progetto di **analisi esplorativa dei dati** di un’azienda agricola, focalizzato sull’analisi di **KPI aziendali**, **vendite**, **produzione** e **clienti**.

Il progetto è stato realizzato in **Python** tramite **Jupyter Notebook** ed è pensato come **progetto da portfolio Data Analyst**, con l’obiettivo di dimostrare competenze di analisi, interpretazione dei dati e comunicazione degli insight.

---

## Obiettivi
- Analizzare le performance aziendali tramite KPI
- Individuare trend e pattern nei dati
- Supportare decisioni di business attraverso l’analisi dei dati

---

## Competenze dimostrate
- Analisi dei dati orientata al business  
- Pulizia e preparazione di dataset reali  
- Analisi e interpretazione di KPI aziendali  
- Utilizzo di Python (Pandas, NumPy) per la data analysis  
- Visualizzazione dei dati per stakeholder non tecnici  
- Strutturazione e documentazione di notebook analitici  

---

## Tecnologie utilizzate
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Jupyter Notebook
